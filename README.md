    <h1><b>Ayrin Fashion Store - eCommerce Website</b></h1>
<h4><b>Overview</b></h4>
<p>Ayrin Fashion Store is a dynamic eCommerce web application developed to offer a seamless online shopping experience. The platform includes various features such as product categorization, pagination, shopping cart functionality, and an efficient search system. Additionally, the website comes with a user-friendly Admin Panel for managing products, categories, and orders.
</p>
<h2>Features</h2>
<ol>
<li>
<b>  Admin Panel</b>
</li>
   <ul>
     <li>Manage products (CRUD)</li>
     <li>Handle product categories</li>
     <li>Monitor and update customer orders</li>
     <li>Secure login for admins</li>
   </ul>
  
<li><b>Product Categorization</b></li>
<ul>
  
<li>Organize products into various categories</li>
<li>Filter products based on categories</li>
</ul>
  
<li><b>Pagination</b></li>

<ul>
 <li> Efficiently display products across multiple pages</li>
<li>Enhance website performance</li>
</ul>

<li><b>Shopping Cart Functionality</b></li>

<ul>
 <li> Add/remove items to/from cart</li>
<li>View and update quantities in the cart</li>
<li>Track total prices before checkout</li>
</ul>

<li><b> Efficient Search System</b></li>

<ul>
  <li>Search for products by name, category, or description</li>
<li>Instant search results display</li>
</ul>
</ol>

<h2>Technologies Used</h2>

<h3>Frontend:</h3>
HTML: Structure of web pages
CSS: Styling and layout design
JavaScript: Interactivity and dynamic content
<h3>Backend:</h3>
Python (Django): Core backend development and functionality implementation
<h3>Database:</h3>
MySQL: Storing product, user, and order information
<h3>Version Control:</h3>
Git: Code versioning and collaboration
